# 使用概览

LeanSpec 提供多种使用Spec的方式，具体取决于您的首选工作流程和用例。

## 快速开始

如果您是 LeanSpec 新手，请从以下内容开始：

1. **[AI 编码工作流](/docs/guide/usage/ai-coding-workflow)** - 学习 AI 辅助开发的推荐工作流
2. **[CLI 使用](/docs/guide/usage/cli/creating-managing)** - 掌握基本的 CLI 命令
3. **[Spec结构](/docs/guide/usage/spec-structure)** - 了解如何构建您的Spec

## 核心工作流

### AI 驱动的开发

LeanSpec 专为 AI 优先的工作流程而设计：

- **[AI 编码工作流](/docs/guide/usage/ai-coding-workflow)** - 推荐的 SDD 工作流
- **[MCP 集成](/docs/guide/usage/mcp-integration)** - 将 AI 代理连接到您的Spec
- **[可视化模式](/docs/guide/visual-mode)** - 在 Web 界面中浏览Spec

### 命令行界面

掌握 CLI 进行日常Spec管理：

- **[创建和管理Spec](/docs/guide/usage/cli/creating-managing)** - 创建、更新和管理Spec
- **[查找Spec](/docs/guide/usage/cli/finding-specs)** - 搜索和发现Spec
- **[项目管理](/docs/guide/usage/cli/project-management)** - 跟踪状态、依赖关系和健康状况

### Spec结构

了解如何编写有效的Spec：

- **[Spec结构](/docs/guide/usage/spec-structure)** - LeanSpec 的解剖结构

## 高级功能

熟悉基础知识后：

- **[模板](/docs/guide/usage/advanced-features/templates)** - 自定义Spec模板
- **[自定义字段](/docs/guide/usage/advanced-features/custom-fields)** - 扩展frontmatter
- **[变量](/docs/guide/usage/advanced-features/variables)** - 使用动态变量
- **[frontmatter指南](/docs/guide/usage/advanced-features/frontmatter)** - 高级frontmatter
- **[代理配置](/docs/guide/usage/advanced-features/agent-configuration)** - 配置 AI 代理

## 选择您的路径

**对于 AI 编码：**
1. 阅读 [AI 编码工作流](/docs/guide/usage/ai-coding-workflow)
2. 设置 [MCP 集成](/docs/guide/usage/mcp-integration)
3. 使用 `lean-spec create` 开始创建Spec

**对于团队管理：**
1. 学习 [CLI 使用](/docs/guide/usage/cli/creating-managing)
2. 使用[可视化模式](/docs/guide/visual-mode)获取概览
3. 使用[项目管理](/docs/guide/usage/cli/project-management)跟踪进度

**对于高级用户：**
1. 掌握所有 CLI 命令
2. 使用[高级功能](/docs/guide/usage/advanced-features/templates)进行自定义
3. 与您的工具链深度集成
