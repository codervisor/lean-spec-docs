# Usage Overview

LeanSpec provides multiple ways to work with specs, depending on your preferred workflow and use case.

## Quick Start

If you're new to LeanSpec, start with:

1. **[AI Coding Workflow](/docs/guide/usage/ai-coding-workflow)** - Learn the recommended workflow for AI-assisted development
2. **[CLI Usage](/docs/guide/usage/cli/creating-managing)** - Master the essential CLI commands
3. **[Spec Structure](/docs/guide/usage/spec-structure)** - Understand how to structure your specs

## Core Workflows

### AI-Powered Development

LeanSpec is designed for AI-first workflows:

- **[AI Coding Workflow](/docs/guide/usage/ai-coding-workflow)** - The recommended SDD workflow
- **[MCP Integration](/docs/guide/usage/mcp-integration)** - Connect AI agents to your specs
- **[Visual Mode](/docs/guide/visual-mode)** - Browse specs in a web interface

### Command Line Interface

Master the CLI for daily spec management:

- **[Creating & Managing Specs](/docs/guide/usage/cli/creating-managing)** - Create, update, and manage specs
- **[Finding Specs](/docs/guide/usage/cli/finding-specs)** - Search and discover specs
- **[Project Management](/docs/guide/usage/cli/project-management)** - Track status, dependencies, and health

### Spec Structure

Understand how to write effective specs:

- **[Spec Structure](/docs/guide/usage/spec-structure)** - Anatomy of a LeanSpec

## Advanced Features

Once you're comfortable with the basics:

- **[Templates](/docs/guide/usage/advanced-features/templates)** - Customize spec templates
- **[Custom Fields](/docs/guide/usage/advanced-features/custom-fields)** - Extend frontmatter
- **[Variables](/docs/guide/usage/advanced-features/variables)** - Use dynamic variables
- **[Frontmatter Guide](/docs/guide/usage/advanced-features/frontmatter)** - Advanced frontmatter
- **[Agent Configuration](/docs/guide/usage/advanced-features/agent-configuration)** - Configure AI agents

## Choose Your Path

**For AI Coding:**
1. Read [AI Coding Workflow](/docs/guide/usage/ai-coding-workflow)
2. Set up [MCP Integration](/docs/guide/usage/mcp-integration)
3. Start creating specs with `lean-spec create`

**For Team Management:**
1. Learn [CLI Usage](/docs/guide/usage/cli/creating-managing)
2. Use [Visual Mode](/docs/guide/visual-mode) for overview
3. Track progress with [Project Management](/docs/guide/usage/cli/project-management)

**For Power Users:**
1. Master all CLI commands
2. Customize with [Advanced Features](/docs/guide/usage/advanced-features/templates)
3. Integrate deeply with your toolchain
