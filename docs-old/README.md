## Quick Reference

### For New Users
1. Read [Main README](../README.md) for quick start
2. Check [Philosophy](PHILOSOPHY.md) to understand the mindset
3. Run `lspec init` and choose a template
4. See [Command Reference](COMMANDS.md) for common commands

### For Teams
1. Review [Integration Guide](INTEGRATION.md) for existing projects
2. Check [Template System](TEMPLATES.md) to choose the right template
3. Read [Frontmatter Spec](FRONTMATTER.md) to configure custom fields
4. Set up [AI Agent Integration](../AGENTS.md) if using AI tools

### For Researchers
1. Read [Comparisons](COMPARISONS.md) to understand positioning
2. Review [Philosophy](PHILOSOPHY.md) for core principles
3. Check [Template System](TEMPLATES.md) to see flexibility options
