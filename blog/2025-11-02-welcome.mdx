# Welcome to LeanSpec!

We're excited to introduce LeanSpec - a lightweight, flexible Spec-Driven Development (SDD) methodology designed for modern AI-powered development.

{/* truncate */}

## What is LeanSpec?

LeanSpec is built on the principle that **AI coding agents need clear context without overwhelming documentation**. Traditional SDD approaches either lack the detail AI needs or create so much overhead that nobody maintains them.
